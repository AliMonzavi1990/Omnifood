## Project Overview
Omnifood is a fully responsive food delivery website that demonstrates modern **frontend development skills**. The project focuses on **UI/UX design, responsive layout, and interactive elements** using HTML, CSS, and JavaScript.  

## Key Features
- **Responsive Design:** Works seamlessly on mobile, tablet, and desktop.  
- **Interactive UI:** Smooth scrolling, animated sections, and hover effects.  
- **Navigation & Forms:** Functional navigation menu and contact/subscription forms.  
- **Modern Styling:** Utilizes CSS Flexbox/Grid, Google Fonts, and Font Awesome icons.  

## Technologies
- HTML5, CSS3, JavaScript (ES6)  
- Google Fonts & Font Awesome  
- Netlify for deployment
